sign up

trash.town is still being set up, but in the mean time, you can prepare for signups before they become available by doing a few things on this page

requirements

to prepare for signups, you will require the following:

  • a unix-like operating system, such as, but not limited to:
    • ubuntu
    • windows subsystem for linux
    • macOS
    • scoop.sh
  • SSH

preparing private and public ssh keys

a public ssh key stays on a server or computer you want to remotely log into. it helps prove who is accesing the server. a private key is like a key to your home. this means you don't want anyone to have access to it or they can access your stuff!

to prepare private and public ssh keys

  1. run ssh-keygen -b 4096 -t rsa -f ~/.ssh/trash.town
  2. press enter, if you don't require a password for access to the key
  3. press enter again to confirm

what did that do?

this generated two things:

  • a ~/.ssh/trash.town key, which is your private key
  • a ~/.ssh/trash.town.pub key, which is your public key

what do i do with these keys?

you will need to send the public key to trash.town admins, so the trash.town ssh server can check if it is you when you ssh into trash.town

how do i send the public key?

try typing cat ~/.ssh/trash.town.pub

a long list of letters and numbers will show up on your terminal, this is the public key you will need to send to the trash.town admins when signups are open

be careful!

make sure you don't send your trash.town key (without the .pub at the end), because this is your super secret private key

if the steps above didn't work for you, have a look at tilde.town's guide or tilde.town's key generator
